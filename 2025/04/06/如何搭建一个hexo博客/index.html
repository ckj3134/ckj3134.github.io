主题

我选择的主题是next
在目录下获取主题
git https://github.com/theme-next/hexo-theme-next themes/next

将它加到submoudle里面不然自动化部署的时候没有next的代码会报错
git submodule add https://github.com/theme-next/hexo-theme-next themes/next
